# productdemo
A app to demo a product master-details view
## Features
### List view
* RecyclerView
* Pull to refresh
* Endless scrolling
* Change to grid view on tablet

### Detail view
* Share to other app
* Swip to previous/next
* Show HTML in description

### Other Features
* Works offline
* Custom view (reusable review UI)
